# はじめに

このリポジトリは、フロントエンドの学習用のリポジトリになります！
まずはこのリポジトリをクローンして、自分の手元の環境で、立ち上げてみましょう！

### 具体的な手順

```bash
# 適切なディレクトリに移動する
# git cloneする
$ git clone git@github.com:aminoray/nuxt3_practice.git

# install
$ npm install

# local server
$ npm run dev
```

localhost は、3030 番ポートを指定しているので、localhost:3030 にアクセスします。
